# Current Project State

Project:
Blade Box Arena

Core state:
- Top-down 2D roguelite/action arena-dungeon game.
- Built with plain HTML, CSS, and JavaScript using HTML5 Canvas.
- Packaged as an Android APK using Capacitor WebView wrapper.
- Android build pipeline works.
- Game installs and runs on Android phone as a normal app.
- Runs in landscape fullscreen.
- Local WiFi co-op works.
- Auto lobby discovery works.
- Manual IP join remains available as fallback.
